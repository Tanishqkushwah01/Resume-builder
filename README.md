# Interview AI (MERN Stack)

An AI-powered platform that analyzes resumes, generates interview questions, identifies skill gaps, and creates optimized resumes tailored to job descriptions.

---

## 🚀 Features

* Upload resume (PDF support)
* Add job description
* AI-generated:

  * Technical interview questions
  * Behavioral interview questions
  * Skill gap analysis
  * Preparation roadmap
* Match score evaluation
* **AI-powered resume generation**
* **Download resume as PDF**

---

## 📄 Resume Generation

* Generates a professional resume using AI
* Tailors resume based on job description
* Improves resume quality and relevance
* Downloadable in PDF format

## ⚙️ Setup Instructions

### 1. Clone the repository

```bash
git clone <your-repo-url>
cd project-folder
```

---

### 2. Install dependencies

```bash
cd Backend
npm install

cd ../Frontend
npm install
```

---

### 3. Create `.env` file (Backend)

```env
JWT_SECRET=hihelloeveryonne
MONGO_URI=your_mongodb_connection
GEMINI_API_KEY=your_api_key
```

---

### 4. Run the project

**Backend**

```bash
npm run dev
```

**Frontend**

```bash
npm run dev
```
